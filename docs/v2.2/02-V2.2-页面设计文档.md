## 2. AI 备课工作台 (`/workspace`)

### 2.1 布局

```
┌──────────────────────────────────────────────────┐
│ TopNav                                              │
├────────────────────────────┬─────────────────────┤
│ 左侧 AI 聊天区 (70%)         │ 右侧工作成果 (30%)    │
│                            │                     │
│ 顶部信息栏                  │ 搜索框               │
│ 文件列表                   │ 类型筛选胶囊          │
│                            │                     │
│ 聊天气泡                   │ 工作卡片             │
│ AI 流式输出                │ 工作卡片             │
│ AI 确认卡片                │                     │
│                            │                     │
│ 上传区+输入框+发送按钮      │                     │
└────────────────────────────┴─────────────────────┘
```

### 2.2 左侧聊天区

- 顶部：当前标题、分类标签、最后编辑时间、原始文件列表（显示真实文件名）
- AI 回复流式输出（SSE，保留普通接口 fallback）
- AI 识别确认卡片：内嵌在聊天气泡中，包含"确认保存"和"修改分类"
- 教学反思关联推荐卡片：匹配度最高的3条个人备课
- 底部：上传区 + 输入框 + 发送按钮

### 2.3 右侧工作成果区

- 搜索框 + 类型筛选胶囊（全部/个人备课/集体备课/教学反思/计划与总结）
- 卡片列表：标题 + 时间 + 类型标签
- 点击整张卡片恢复聊天

## 3. 首页 (`/home`)

- 保留按教研组展示教师的卡片式布局
- 教师姓名区域精简：首字头像 + 姓名 + 备课数/反思数

---

## 4. 业务页面通用结构

```
┌──────────────────────────────────────────────┐
│ 页面标题 + 筛选栏（学年学期/教师/关键词）       │
├──────────────────────────────────────────────┤
│ 数据表格（现代化样式）                          │
│ 序号 | 关键字段 | 创建时间 | 操作               │
│ → 点击操作打开右侧 Drawer 详情                  │
│   基本信息 + 文件预览 + Timeline               │
└──────────────────────────────────────────────┘
```

---

## 5. Drawer 规范

- 使用 shadcn/ui Sheet
- 宽度 720px（普通详情）/ 840px（文件预览）
- ESC / 点击遮罩关闭

## 6. 类型标签颜色

| 类型       | 颜色                              |
| ---------- | --------------------------------- |
| 个人备课   | 蓝 `bg-blue-50 text-blue-700`     |
| 集体备课   | 绿 `bg-green-50 text-green-700`   |
| 教学反思   | 橙 `bg-orange-50 text-orange-700` |
| 计划与总结 | 紫 `bg-purple-50 text-purple-700` |

## 7. AI 配置后台 (`/admin/ai-config`)

- 管理员可访问，教师不可见
- Provider 管理 + API Key（加密存储）+ Base URL + 默认模型 + 连接测试 + Token 统计
