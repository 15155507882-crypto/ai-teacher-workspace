## 2. AI 备课工作台 (`/workspace`)

### 2.1 布局

```
┌──────────────────────────────────────────────────┐
│ TopNav：学校/用户/设置                              │
├────────────────────────────┬─────────────────────┤
│ 左侧 AI 聊天区 (70%)         │ 右侧工作成果 (30%)    │
│                            │                     │
│ 顶部信息栏                  │ 搜索框               │
│ 文件列表                   │ 类型筛选胶囊          │
│                            │                     │
│ 聊天气泡                   │ 工作卡片             │
│ AI 流式输出                │ 工作卡片             │
│ AI 确认卡片                │ 工作卡片             │
│                            │                     │
│ 上传区+输入框+发送按钮      │                     │
└────────────────────────────┴─────────────────────┘
```

### 2.2 左侧聊天区

- 顶部：当前标题、分类标签、最后编辑时间、原始文件列表（显示真实文件名）
- AI 回复流式输出
- AI 识别确认卡片：内嵌在聊天气泡中，包含"确认保存"和"修改分类"按钮
- 教学反思关联推荐卡片：显示匹配度最高的3条个人备课，教师选择关联
- 底部：上传区 + 输入框 + 发送按钮

### 2.3 右侧工作成果区

- 搜索框 + 类型筛选胶囊（全部/个人备课/集体备课/教学反思/计划与总结）
- 卡片列表：标题、时间、类型标签
- 点击整张卡片恢复聊天

## 4. 业务页面通用结构

四个业务页面（集体备课/个人备课/计划总结/教学反思）统一结构：

```
┌──────────────────────────────────────────────┐
│ 页面标题                                      │
├──────────────────────────────────────────────┤
│ 筛选栏：学年学期/教师/关键词                    │
├──────────────────────────────────────────────┤
│ 数据表格（现代化样式，非密集后台表）              │
│ 序号 | 关键字段 | 创建时间 | 操作               │
├──────────────────────────────────────────────┤
│ 点击操作→右侧 Drawer 详情                      │
│   - 基本字段                                   │
│   - 文件在线预览                               │
│   - 参与/反思 Timeline                         │
└──────────────────────────────────────────────┘
```

---

## 5. Drawer 规范

- 所有详情使用右侧滑入 Drawer
- 宽度 720px（普通详情）/ 840px（文件预览）
- ESC 关闭、点击遮罩关闭
- 内部滚动
