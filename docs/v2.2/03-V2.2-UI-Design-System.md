# V2.2 UI Design System

## 1. 组件库选型

- **基础组件**：shadcn/ui + Radix UI（优先，不纯手写 Tailwind）
- **图标**：Lucide Icons（统一，不混用）
- **动画**：Framer Motion（可选）/ Tailwind animate
- **表格**：shadcn/ui Table
- **Drawer**：shadcn/ui Sheet

## 2. 色彩规范

| Token              | HEX       | 用途             |
| ------------------ | --------- | ---------------- |
| `--primary`        | `#2563EB` | 主色、按钮、链接 |
| `--primary-hover`  | `#1D4ED8` | Hover            |
| `--primary-light`  | `#DBEAFE` | 浅底             |
| `--success`        | `#22C55E` | 成功             |
| `--warning`        | `#F59E0B` | 警告             |
| `--danger`         | `#EF4444` | 危险/删除        |
