# V2.2 UI Design System

## 1. 组件库

- **基础组件**：shadcn/ui + Radix UI
- **图标**：Lucide Icons（统一，不混用）
- **表格**：shadcn/ui Table
- **Drawer**：shadcn/ui Sheet

## 2. 色彩

| Token         | HEX       | 用途 |
| ------------- | --------- | ---- |
| Primary       | `#2563EB` | 主色 |
| Primary Hover | `#1D4ED8` |      |
| Primary Light | `#DBEAFE` |      |
| Success       | `#22C55E` |      |
| Warning       | `#F59E0B` |      |
| Danger        | `#EF4444` |      |
| Page BG       | `#F8FAFC` |      |

| Muted BG      | `#F1F5F9` |      |
| Title         | `#0F172A` |      |

## 3. 类型标签

| 类型       | 样式                           |
| ---------- | ------------------------------ |
| 个人备课   | `bg-blue-50 text-blue-700`     |
| 集体备课   | `bg-green-50 text-green-700`   |
| 教学反思   | `bg-orange-50 text-orange-700` |
| 计划与总结 | `bg-purple-50 text-purple-700` |

## 4. 间距（8px 栅格）

| Token | px  |
| ----- | --- |
| xs    | 4   |
| sm    | 8   |
| md    | 16  |
| lg    | 24  |

## 5. 圆角

小组件 8px / 卡片 16px / Drawer 20px（左上左下）

## 6. 按钮高度

sm 32px / md 40px / lg 48px

## 7. 字体

中文：PingFang SC / Microsoft YaHei
英文/数字：Inter
层级：H1 24-28px / H2 20-22px / Title 16-18px / Body 14px / Caption 12px
