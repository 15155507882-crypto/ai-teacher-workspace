# V2.2 数据库变更文档

## 1. 变更原则

- 基于 V2.1 现有表增量调整
- 优先在原表增加字段
- 关联复杂时新增关联表
- 所有新字段允许 NULL 或有默认值
- 生成 Migration 文件，支持回滚

## 2. 现有表复用

| V2.1 表           | V2.2 用途                                            |
| ----------------- | ---------------------------------------------------- |
| `content`         | **工作记录主表**（右侧成果列表、聊天恢复、分类记录） |
| `personal_lesson` | 个人备课详情（增加 week_no）                         |
| `group_lesson`    | 集体备课详情（增加 week_no, group_lesson_type）      |
| `reflection`      | 教学反思（增加 match_score）                         |
| `plan_summary`    | 计划与总结（不变）                                   |
| `file_asset`      | 文件（增加 parsed_text, preview_status）             |
| `ai_message`      | 聊天消息（复用，role=user/assistant）                |
| `ai_session`      | 聊天会话（复用）                                     |

## 4. 现有表新增字段

```sql
-- content 无需新增字段（已包含 title/category/summary）
-- personal_lesson 新增
ALTER TABLE personal_lesson ADD COLUMN week_no int DEFAULT NULL;

-- group_lesson 新增
ALTER TABLE group_lesson ADD COLUMN week_no int DEFAULT NULL;
ALTER TABLE group_lesson ADD COLUMN group_lesson_type varchar(30) DEFAULT NULL;

-- reflection 新增
ALTER TABLE reflection ADD COLUMN match_score decimal(5,2) DEFAULT 0;

-- file_asset 新增
ALTER TABLE file_asset ADD COLUMN parsed_text text DEFAULT NULL;
ALTER TABLE file_asset ADD COLUMN preview_status varchar(20) DEFAULT 'none';
```
