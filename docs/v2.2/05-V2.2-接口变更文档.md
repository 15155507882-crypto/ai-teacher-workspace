# V2.2 接口变更文档

## 1. 变更原则

- 基于 V2.1 现有接口增量调整
- 优先复用已有接口
- 新增接口遵循 RESTful 规范
- 所有接口校验：登录状态、角色、学校隔离

## 2. 可复用接口

| V2.1 接口                             | V2.2 用途               |
| ------------------------------------- | ----------------------- |
| `POST /api/auth/login`                | 登录（含验证码）        |
| `GET /api/auth/me`                    | 当前用户信息            |
| `GET /api/public/school`              | 学校信息                |
| `POST /api/ai/chat`                   | AI聊天（增强为SSE流式） |
| `GET /api/ai/recognition/:id`         | 轮询AI结果              |
| `POST /api/ai/confirm`                | 确认保存                |
| `POST /api/ai/upload`                 | 文件上传                |
| `POST /api/ai/undo/:id`               | 撤销                    |
| `GET /api/teachers/:id/contents`      | 教师内容列表            |
| `GET /api/teachers/:id/content-stats` | 内容统计                |
| `GET /api/contents/:id`               | 内容详情                |
| `DELETE /api/contents/:id`            | 删除内容                |
| `GET /api/admin/teachers`             | 教师管理列表            |
| `POST /api/admin/teachers`            | 新增教师                |
| `PUT /api/admin/teachers/:id`         | 编辑教师                |
| `GET /api/admin/departments`          | 组织列表                |

## 3. 需新增接口

### AI 配置

| 方法   | 路径                            | 说明             |
| ------ | ------------------------------- | ---------------- |
| `GET`  | `/api/admin/ai/providers`       | 获取Provider列表 |
| `POST` | `/api/admin/ai/school-config`   | 保存学校AI配置   |
| `POST` | `/api/admin/ai/test-connection` | 测试连接         |
| `GET`  | `/api/admin/ai/token-stats`     | Token统计        |

### 业务页面列表

| 方法   | 路径                        | 说明             |
| ------ | --------------------------- | ---------------- |
| `GET`  | `/api/group-lessons`        | 集体备课列表     |
| `GET`  | `/api/group-lessons/:id`    | 集体备课详情     |
| `GET`  | `/api/personal-lessons`     | 个人备课列表     |
| `GET`  | `/api/personal-lessons/:id` | 个人备课详情     |
| `GET`  | `/api/reflections`          | 教学反思列表     |
| `GET`  | `/api/reflections/:id`      | 教学反思详情     |
| `POST` | `/api/reflections/match`    | 推荐关联个人备课 |
| `GET`  | `/api/plan-summaries`       | 计划总结列表     |
| `GET`  | `/api/plan-summaries/:id`   | 计划总结详情     |

### 文件预览

| 方法  | 路径                      | 说明             |
| ----- | ------------------------- | ---------------- |
| `GET` | `/api/files/:id/preview`  | 获取文件预览内容 |
| `GET` | `/api/files/:id/download` | 下载文件         |

## 4. 需调整接口

| 接口                                  | 调整项                               |
| ------------------------------------- | ------------------------------------ |
| `POST /api/ai/chat`                   | 支持 SSE 流式输出                    |
| `GET /api/ai/recognition/:id`         | 响应增加 `nl_reply` 字段             |
| `GET /api/teachers/:id/contents`      | 增加筛选参数 academic_year、semester |
| `GET /api/teachers/:id/content-stats` | 响应增加 academic_year、semester     |
