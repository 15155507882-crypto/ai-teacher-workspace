# V2.2 开发任务拆分

## 优先級说明

- **P0**：阻塞项，必须最先完成
- **P1**：核心功能，第二阶段完成
- **P2**：增强项，第三阶段完成

---

## P0（第一阶段）

### T-001：数据库增量迁移

- 新增 `ai_providers`、`school_ai_configs`、`ai_call_logs` 三张表
- 现有表增加字段（personal_lesson/week_no, group_lesson/week_no+type, reflection/match_score, file_asset/parsed_text+preview_status）
- 生成 Migration 文件
- 提供回滚脚本
- **依赖**：无
- **验收**：Migration 执行成功，新表创建，新字段存在

### T-002：shadcn/ui 基础组件落地

- 安装 shadcn/ui + Radix UI + Lucide Icons
- 实现/统一：Button, Input, Select, Card, Table, Badge, Drawer(Sheet), Upload, Toast, Skeleton
- 统一样式 Token（颜色/字体/间距/圆角）
- **依赖**：无
- **验收**：所有组件 Storybook 可查看

### T-003：AI 配置后台

- 新增 `/admin/ai-config` 页面
- Provider 管理（超级管理员维护）
- 学校 AI 配置（API Key 加密存储、Base URL、默认模型）
- 连接测试（保存后立即测试）
- Token 统计（今日/本周/本月/累计）
- 权限：仅管理员可见，教师不可见
- **依赖**：T-001, T-002
- **验收**：CRUD + 连接测试 + Token统计正常

### T-004：AI 工作台两栏改造

- 基于现有 `/workspace` 页面进行 UI 与交互重构
- 三栏 → 两栏：左侧 70% AI 聊天区 + 右侧 30% 工作成果卡片列表
- 左侧：上传区、聊天区（含 AI 确认卡片）、顶部信息栏
- 右侧：搜索框、类型筛选胶囊、工作记录卡片（点击恢复聊天）
- SSE 流式输出（保留普通接口 fallback）
- **依赖**：T-002
- **验收**：聊天→识别→确认→保存→右侧列表刷新，完整流程通过

### T-005：首页精简

- 保留教研组教师列表卡片式布局
- 精简教师姓名区域：首字头像 + 姓名 + 备课数/反思数
- **依赖**：无
- **验收**：首页按教研组展示，教师区域简洁
